# SPEC-UI-001 인수 기준 (Acceptance Criteria)

## 검증 방법

Streamlit 앱을 `uv run streamlit run streamlit_app.py`로 실행한 후, FastAPI 서버(`uv run uvicorn app.main:app --reload`)가 함께 기동된 상태에서 아래 시나리오를 수동 또는 자동화 테스트로 검증한다.

---

## AC-UI-001: 양력 입력으로 사주 계산 성공

**관련 요구사항**: REQ-UI-001, REQ-UI-005, REQ-UI-010, REQ-UI-011

```gherkin
Given 사용자가 Streamlit 앱을 실행한 상태이고
  And 달력 타입을 "양력"으로 선택하고
  And 년/월/일을 유효한 날짜(예: 1984/04/15)로 입력하고
  And 시각 슬라이더를 14시로 설정하고
  And 성별을 "남성"으로 선택했을 때
When 사용자가 "사주 계산" 버튼을 클릭하면
Then 로딩 스피너가 표시된 후 계산이 완료되어야 한다
  And 탭 4개 (사주 원국, 십성 분석, 운의 흐름, 세부 지표)가 표시되어야 한다
  And "사주 원국" 탭에 4기둥(年柱·月柱·日柱·時柱) 카드가 표시되어야 한다
  And 각 기둥 카드 아래에 상징 의미(조상·어린시절 등)가 표시되어야 한다
```

## AC-UI-002: 시주 미상 처리

**관련 요구사항**: REQ-UI-003, REQ-UI-012, REQ-UI-023, REQ-UI-042

```gherkin
Given 사용자가 입력 폼에서
  And "시각 미상" 체크박스를 선택한 상태일 때
When 사용자가 "사주 계산" 버튼을 클릭하면
Then API 요청의 birth_hour 필드가 null로 전송되어야 한다
  And "사주 원국" 탭의 時柱 카드에는 간지 대신 "미상"이 표시되어야 한다
  And 時柱 카드의 상징 의미(자녀·말년)는 정상 표시되어야 한다
  And "십성 분석" 탭의 시간·시지 행은 "미상"으로 표시되어야 한다
  And "세부 지표" 탭의 시지 십이운성 셀은 "미상"으로 표시되어야 한다
```

## AC-UI-003: 음력 + 윤달 입력

**관련 요구사항**: REQ-UI-002, REQ-UI-004

```gherkin
Given 사용자가 달력 타입을 "음력"으로 선택했을 때
When 입력 폼이 렌더링되면
Then 사이드바에 "윤달" 체크박스가 표시되어야 한다
  And 달력 타입을 다시 "양력"으로 변경하면 윤달 체크박스가 숨겨져야 한다

Given 사용자가 음력·윤달 체크박스 모두 선택하고 유효한 날짜를 입력했을 때
When "사주 계산" 버튼을 클릭하면
Then API 요청에 is_lunar=true, is_leap_month=true가 포함되어야 한다
  And 계산 결과가 정상 표시되어야 한다
```

## AC-UI-005: 십성 분석 탭 정상 표시

**관련 요구사항**: REQ-UI-020, REQ-UI-021, REQ-UI-022, REQ-UI-024

```gherkin
Given 사주 계산이 완료된 상태에서
When 사용자가 "십성 분석" 탭을 클릭하면
Then 일간(日干) 정보가 탭 상단에 표시되어야 한다
  And 나머지 7글자(또는 시주 미상 시 5글자)의 십성 테이블이 표시되어야 한다
  And 테이블에는 위치, 간지, 십성명, 그룹 열이 포함되어야 한다
  And 비겁·식상·재성·관성·인성 5개 그룹 설명이 expander로 표시되어야 한다
  And expander를 클릭하면 해당 그룹의 성격·사회적 의미가 펼쳐져야 한다
```

## AC-UI-011: API 오류 처리 — 네트워크 오류

**관련 요구사항**: REQ-UI-050, REQ-UI-053

```gherkin
Given FastAPI 서버가 기동되지 않은 상태에서
When 사용자가 "사주 계산" 버튼을 클릭하면
Then "API 서버에 연결할 수 없습니다" 또는 "요청 시간이 초과되었습니다" 오류 메시지가 표시되어야 한다
  And Python 예외 스택 트레이스가 화면에 표시되지 않아야 한다
  And 탭 영역은 표시되지 않아야 한다
```
